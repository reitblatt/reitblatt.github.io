# Notes for myself

The website is generated with the Jekyll static site generator.

## Instructions

Install dependencies:

```
bundle
```

Build the site:

```
bundle exec jekyll build
```

Test locally:

```
bundle exec jekyll serve
```

To update the live site, git commit && git push
